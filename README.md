# Global Bond Market Dashboard

A BondStats signature dashboard for tracking sovereign yields, yield curve conditions, recession-style indicators, real yields and liquidity-sensitive bond market regimes.

## Features

- Sovereign yield snapshot
- Yield curve and recession signals
- Real yield indicator
- Liquidity and Basel demand inputs
- Regime classification engine
- Interpretation panel
- Scenario matrix
- Professional dashboard layout
